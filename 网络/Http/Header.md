# HTTP Header
> Http Header有四种： **通用header**、**请求header**、**响应header**、**实体header**

### 通用首部

|    名称              |     含义               |
| ------------------- | ---------------------- |
|   Cache-Control     |  控制缓存的行为           |
|   Connection        |  逐跳首部、连接的管理      |
|   Date              |  创建报文的时间           |
|   Pragma            |  报文指令                |
|   Trailer           |  报文末端的首部一览        |
|   Transfer-Encoding |  指定报文主体的传输编码方式 |
|   Upgrade           |  升级为其它协议           |
|   Via               |  代理服务器的相关信息      |
|   Warning           |  错误通知                |
